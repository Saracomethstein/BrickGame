Introduction

This game, developed in C, is a classic brick game where the objective is to clear rows of bricks by controlling a falling block.

Key Features:

  • Interactive terminal interface using ncurses
  • Score tracking and high score saving
  • Various levels of difficulty

Setup & Launch

Dependencies:

This game requires the following libraries:

  • ncurses

Compilation:

To install the game, use the following command:

# make install

Running the Game:

Once compiled, you can run the game with:

# make run

Game Architecture

The game is built around a central loop that handles game state updates, rendering, and user input.

Main Components:

  • GameInfo_t: Structure holding game state
  • WinStruct: Structure for window management
  • ncurses: Library for terminal UI

User Guide

Controls:

  • Arrow Keys: Move block left, right or down
  • Spacebar: Rotate block
  • q: Quit game
  • r: Restart game
  • p: Pause game

Objective:

Clear rows of bricks by strategically placing blocks.

Additional Information

Bugs & Troubleshooting:

Known issues include potential crashes on terminal resize. To resolve, avoid resizing the terminal during play.
